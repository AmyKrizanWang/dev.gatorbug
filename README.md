# dev.gatorbug

## Environment: DevEnv

This environment is a **DevEnv** used only for testing plugins and extensions.

### Important Notice

⚠️ **No production deployments are allowed from this environment.**

All changes must pass QA in the **StageEnv** before they can be deployed to production.

### Workflow

1. Develop and test plugins/extensions in this DevEnv
2. Submit changes for QA review in StageEnv
3. Only after passing QA in StageEnv can changes be deployed to production
